ANodeJsWebSite
==============

the first time try to build a web site by nodejs

## 项目介绍
	最近大概的学习了一下nodejs，可是学习总要致用才算有用，所以就想建立一个以nodejs为后台的简单的web站点。
## 功能介绍
	在这个项目中，通过nodejs控制浏览器对服务器的请求，展示对应的界面或者对Ajax的请求，处理并返回对应的参数
